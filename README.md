# bract.ring

A [Bract](https://github.com/bract) module for [Ring](https://github.com/ring-clojure) support.


## Usage

Leiningen coordinates: `[bract/bract.ring "0.3.0-SNAPSHOT"]`

For documentation refer demo applications.
